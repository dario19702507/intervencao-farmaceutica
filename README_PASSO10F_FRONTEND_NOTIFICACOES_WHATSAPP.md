# Passo 10F — Frontend de Notificações e WhatsApp

Substitua/adicone somente os arquivos do frontend.

Arquivos incluídos:

- frontend/src/App.jsx
- frontend/src/components/layout/Sidebar.jsx
- frontend/src/pages/notificacoes/NotificacoesWhatsapp.jsx
- frontend/src/pages/notificacoes/NotificacoesWhatsapp.css

Funcionalidades:

- Dashboard de notificações internas.
- Lista de notificações com filtros.
- Marcar notificação como lida.
- Marcar todas como lidas.
- Gerar notificações automáticas.
- Enfileirar notificações para WhatsApp.
- Dashboard da fila WhatsApp.
- Fila de envios com filtros.
- Envio manual para fila.
- Simulação de envio.
- Cancelamento e reenfileiramento.

Observação: esta versão ainda usa a camada de simulação do backend. Não realiza envio externo por API real.
